Choose how you want to distribute your password generator to customers:

üåê Web Version (Easiest & Free)

Host the web version online so customers can use it in their browser.

Option A: GitHub Pages (Free)

  1. Create a GitHub account and repository
  2. Upload these files: index.html, style.css, script.js
  3. Enable GitHub Pages in repository settings
  4. Your app will be live at: https://yourusername.github.io/password-generator
GitHub Pages Guide

Option B: Netlify (Free)

  1. Go to netlify.com
  2. Drag your project folder to deploy
  3. Get instant URL like: https://amazing-app-123.netlify.app
Try Netlify

üíª Desktop Apps

Create downloadable applications for Windows, Mac, and Linux.

npm install npm run build

This creates installer files in the dist/ folder:

  • Windows: .exe installer
  • Mac: .dmg file
  • Linux: .AppImage file

You can then:

  • Host download links on your website
  • Upload to GitHub Releases
  • Submit to app stores

üì± Progressive Web App (PWA)

Let users "install" your web app to their device like a native app.

Add a manifest file to enable PWA features:

{ "name": "Password Generator", "short_name": "PassGen", "start_url": "./", "display": "standalone", "background_color": "#667eea", "theme_color": "#667eea", "icons": [ { "src": "icon-192.png", "sizes": "192x192", "type": "image/png" } ] }
